Elszr is szksg van a sebessg idfggsre. Ezt a magassg$--$id fggvny numerikus derivltjval fogjuk most meghatrozni. Ha egy $y(x)$ fggvnyt mintavtelezsvel vges darab $x_i,y_i$ prt kapunk, akkor az $y(x)$ fggvny numerikus derivltjt a kvetkez differencia hnyadossal kzelthetjk:

$$\left . \frac{\mathrm{d}y}{\mathrm{d}x}\right|_{x_i} =\frac{y_{i+1}-y_i}{x_{i+1}-x_i} $$

Definiljunk most egy fggvnyt, mely kt ugyanolyan hosszsg tmb x s y adataibl legenerlja a kt tmb numerikus derviltjt. Figyeljk meg, hogy az utols s az els adatpontban a fenti kplettl egy kicsit eltr mdon jrunk el.

# numerikus derivlt fggvny
def nderiv(y,x):
    "Els szomszd differencil"
    n = len(y) # adatpontok szma
    d = zeros(n) # vltoz inicializls. A zeros() fggvny tetszleges alak s 0-kat tartalmaz arrayt gyrt
    # mivel a legegyszerbb numerikus differencils nem szimmetrikus a vgpontokat
    # kicsit mskpp kezeljk mint a tmb belsejben lv pontokat
    for i in range(1,n-1):
        d[i] = (y[i+1]-y[i])/(x[i+1]-x[i])   #egy ltalnos pont derivltja
    d[0] = (y[1]-y[0])/(x[1]-x[0])           #az els pont derivltja
    d[n-1] = (y[n-1]-y[n-2])/(x[n-1]-x[n-2]) # az utols pont derivltja
    return d
