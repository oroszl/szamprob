Modulok: amikor egy hétköznapi felhasználó óriások vállán áll¶

A Pythonban, mint minden ms modern nyelvben, szoks elre megrt, hasznos fggvnyeket s fggvnycsomagokat hasznlni. A Python szintaxisban az import parancs segtsgvel tudunk kls fggvnyeket, csomagokat vagy modulokat betlteni. Modulnak neveznk egy Python nyelven rt file-t, ami Python-fggvnyek, esetleg -osztlyok definciit tartalmazza. Rendszerint ez egy .py kiterjeszs file. Tbb ilyen defincikat tartalmaz file-ok sszessgt hvjuk csomagnak. Nagy, sok fggvnydefincit tartalmaz, csomagokat szoks alcsomagokra osztani. Ebben a notebookban a numpy a matplotlib s a csv csomagok nhny hasznos fggvnyvel s adatstruktrjval fogunk megismerkedni.
Mieltt ezeket hasznlni tudnnk be kell ket tlteni. Lssunk erre egy pldt!
